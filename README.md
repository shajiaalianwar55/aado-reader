# Aado Reader

A mobile PDF reader focused on a calm, comfortable reading experience.

## Features (roadmap)

- Open PDFs from your device and keep a recent library
- Smooth page navigation with zoom and fit modes
- Day, sepia, and night reading themes
- Search, bookmarks, and restore last reading position

## Getting started

```bash
npm install
npx expo start
```

Then open the project in Expo Go (Android/iOS) or a simulator.
